☕️ TeapotApps Framework

GitHub

A lightweight Node.js framework with modern ESM support

Build scalable backend applications and APIs with ease using a clean, modular architecture.

✨ Features

  • ⚡ Native ESM support (modern JavaScript)
  • 📦 Modular architecture: controllers, routes, services
  • ⚙️ Easy setup and configuration
  • 🛡️ Middleware support (CORS, logging, etc.)
  • 📈 Scalable and maintainable codebase

📦 Installation

Install the CLI globally and scaffold a new project:

npm install -g teapotapps
create-teapotapps my-app

Or use the alias:

teapotapps create my-app

▶️ Usage

cd my-app
teapotapps dev

The development server runs at: http://localhost:3010
